boss_dr12

Compute the likelihood of the supplied expansion and growth history against BOSS DR12 data

File

likelihood/boss_dr12/boss_dr12.py

Attribution

CosmoSIS Team

URL

https://www.sdss3.org/science/boss_publications.php

Citations

arxiv: 1607.03155

This module gives a likelihood of the comoving angular diameter distance D_m, the Hubble parameter H(z) and f*sigma_8. It uses the sound horizon at last-scatter rs_zdrag. A correlated Gaussian likelihood is then returned.

Assumptions

  • LCDM Model

  • Details of BOSS reconstruction
